# StudyBuddyAi
# Learning Management System (LMS)

A full-stack Learning Management System developed during my Software Development Internship at Signity Software Solutions Pvt. Ltd.

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy ORM
- Next.js
- JWT Authentication
- Bcrypt Password Hashing

## Features

- User Registration & Login
- JWT-based Authentication
- Role-Based Access Control (Admin & Student)
- Course Management
- Course Content Delivery
- RESTful APIs

## Status

Currently under active development.
